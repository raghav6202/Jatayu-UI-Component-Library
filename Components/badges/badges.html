Badges

Profile Badges are used to show the current status of the user

Profile Badges and Numbered Badges

Badges are available in 2 different types . The first type displays the status of the user.The second type is used to show the notification count.You need to include badge class for profile status and depending on the status you can use online, offline and dnd classes.If you want the notification badge you can use icon-badge class (e.g. class="badge" , class="icon-badge" )
