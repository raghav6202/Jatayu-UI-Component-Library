Avatar

Avatar component is used to show user's profile picture on profile information page, on navigation bar, in blogs grid items.

Image Avatar

Avatar is available in 4 different sizes. You can use image in Avatar. You need to include these classes in the image tag.Choose the appropriate classes according to the required size img-avatar xl, img-avatar l, img-avatar m, img-avatar s (e.g. class="img-avatar m") Don't forget to add Responsive Image class names for img element.
